Planning and Decision-Making in Groups
Level: A1
Lesson Type: Can Do

Sentence Structure: "I think the best approach is [decision] because [reason]."

Pre Test Questions:

1. "I think the best approach is to assign one lead per task because it avoids confusion."
2. "I think the best approach is to meet once a week because it keeps everyone on the same page."
3. "I think the best approach is to vote on the final idea because it gives everyone a fair say."
4. "I think the best approach is to start with the most urgent tasks because it helps us meet the deadline."
5. "I think the best approach is to create a shared timeline because it helps the team stay organised."
6. "I think the best approach is to divide the work by strengths because it improves overall quality."
7. "I think the best approach is to set a clear agenda before every meeting because it saves time."
8. "I think the best approach is to review progress every Friday because it helps us catch issues early."
9. "I think the best approach is to use a group chat for updates because it keeps everyone informed."
10. "I think the best approach is to finalise the plan before execution because it reduces last-minute changes."

Post Test Questions:

1. "I think the best approach is to pilot the idea with a small group because it reduces risk."
2. "I think the best approach is to gather everyone's input first because it leads to better decisions."
3. "I think the best approach is to break the project into phases because it makes progress easier to manage."
4. "I think the best approach is to document all decisions because it avoids future misunderstandings."
5. "I think the best approach is to set individual accountability for each task because it improves ownership."
6. "I think the best approach is to have a backup plan because it prepares the team for unexpected changes."
7. "I think the best approach is to prioritise customer needs because it drives the most impact."
8. "I think the best approach is to complete the research before deciding because it ensures an informed choice."
9. "I think the best approach is to hold a brainstorming session first because it generates more creative ideas."
10. "I think the best approach is to evaluate all options before committing because it leads to a stronger decision."
